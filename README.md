# kursovaya

## Регистрация и вход

1. На стартовом экране откройте форму **«Регистрация»**.
2. Введите логин и пароль (оба поля обязательны).
3. Нажмите **«Создать аккаунт»**.
4. После успешной регистрации система автоматически выполнит вход и откроет главную страницу пользователя.

Если аккаунт уже есть:

1. Откройте форму **«Вход»**.
2. Введите логин и пароль.
3. Нажмите **«Войти»**.

## Доступ к разделам

До авторизации доступны только страницы **входа и регистрации**.
Разделы **«Мои проекты»**, **«Конструктор»** и **«База пряжи»** открываются только после входа в аккаунт.
